## 使用方式
- 要安装的依赖 PyOpenGl PyOpenGl_accelerate
- 通过 pycharm 添加依赖: setting - project - project inter
- 窗口可以绘制，但有以下报错
```
Unable to load numpy_formathandler accelerator from OpenGL_accelerate
```
